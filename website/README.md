# CNDF 2023 sample website

## Setup

```shell
# in this directory
$ bundle install
$ npm install
```

## Build

```shell
$ bundle exec middleman build
```

## Deploy

### Firebase Hosting

```shell
# in this directory
$ bundle exec middleman build
$ npx firebase login
$ npx firebase deploy
```
